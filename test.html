Program Description

We are going to move away from input via "prompt" and output via "alert". Those are fine for smaller interactive programs, or for occasional use. But for many web pages input comes from text fields on the page. Often output is reported similarly, in text fields on the page. (Another way is to write a completely new web page, but we will save that for later).

In other programming languages it is not unusual for input data to come from files stored on the hard drive. JavaScript can do this too, but many JavaScript programs are intended for use in Webpages, so file use is not nearly as common in JavaScript as in other languages.

In this program we will type input values into text fields on the web page (see above). Then when the "run" button is clicked, the values we typed will be transmitted to your JavaScript main function as arguments. This means your program will have some things to do right at the beginning, no matter what else it will eventually do:

  • validate each input argument to make sure it is the right type, range, and format
  • form an appropriate response and send it back to the web page (via "return")
